PureGym Model Context Protocol server

puregym-mcp is a Python package and MCP server for browsing PureGym centers, discovering classes, checking your bookings, and managing bookings through MCP-compatible clients such as OpenCode and VS Code.

uvx puregym-mcp --transport stdio package_2 PyPI package

Capabilities

The server exposes a compact set of tools for public class discovery and optional authenticated booking actions.

search Class discovery

Read-only tools available without PureGym credentials.

  • • get_capabilities
  • • list_class_types
  • • list_centers
  • • search_classes

event_available Booking management

Authenticated tools for your own account when credentials are configured.

  • • list_my_bookings
  • • book_class
  • • cancel_booking

Modes

Anonymous mode is intended for public class discovery and exposes read-only tools with a 14-day search window.

Authenticated mode is enabled when PUREGYM_USERNAME and PUREGYM_PASSWORD are present. It unlocks booking tools and expands the default search window to 28 days.

Quickstart

The easiest path for most users today is local stdio usage from an MCP-compatible client.

install 
uv tool install puregym-mcp
uvx puregym-mcp --transport stdio
authenticated-local-run 
PUREGYM_USERNAME="your-email" \
PUREGYM_PASSWORD="your-password" \
puregym-mcp --transport stdio

MCP Configuration

Add the server to your MCP client configuration with a simple stdio command. Remote HTTP clients can use the hosted endpoint at /mcp.

OpenCode / Claude Desktop style config

{
  "mcpServers": {
    "puregym": {
      "command": "puregym-mcp",
      "args": ["--transport", "stdio"],
      "env": {
        "PUREGYM_USERNAME": "your-email",
        "PUREGYM_PASSWORD": "your-password"
      }
    }
  }
}

VS Code mcp.json

{
  "servers": {
    "puregym": {
      "type": "stdio",
      "command": "puregym-mcp",
      "args": ["--transport", "stdio"]
    }
  }
}

Deployment

For self-hosted remote usage, run the server over streamable HTTP and expose it at a stable URL. The endpoint on this host is available at https://mcp.jorgesintes.dev/mcp.

streamable-http 
puregym-mcp --transport streamable-http --host 0.0.0.0 --port 8000 --streamable-http-path /mcp
